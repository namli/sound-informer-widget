Sound Informer Widget - Test Page

О виджете

Sound Informer Widget — встраиваемый виджет для отображения интерактивной карты с точками расположения датчиков мониторинга шумового загрязнения. Датчики расположены в трех городах: Париж, Барселона и Дахаб.

⚠️ Важно для работы виджета:

Для работы виджета требуется действующий Mapbox Public Token с настроенными URL restrictions. В текущей демо-версии используется тестовый токен Mapbox, который может не работать. Для полноценной работы замените токен на свой в коде ниже.
